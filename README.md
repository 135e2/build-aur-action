# build-aur-action

## Example usage
```yaml
uses: 135e2/build-aur-action@master
with:
  repo-name: qv2ray-dev-git
```

## Inputs
### `repo-name`
**Required** The name of the AUR repo to build. Default `"qv2ray-dev-git"`.
